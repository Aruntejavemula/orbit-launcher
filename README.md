# Orbit Launcher

A sleek React + TypeScript dashboard for tracking app subscriptions, trials, and renewals. Built with Vite, Tailwind CSS, and Lucide icons.

## Features

- **App Grid** — Visual overview of all your tools with smart urgency badges
- **Quick Add** — Add apps manually with custom icons, categories, and billing details
- **Calendar View** — See upcoming renewals and trial expirations at a glance
- **Insights & Usage** — Track spending and tool adoption
- **Settings & Profile** — Edit your profile, pick themes, and learn the app via an interactive tutorial
- **Dark Mode Ready** — Fully styled for light and dark themes

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Lucide React
- LocalStorage persistence (no backend required)

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```
